How to Create a New Environment

Follow these steps to add a new environment:

  1. Add a new entry in the JSON file under the "ambienti" section.
  2. Specify the environment name and the path to the background image.
  3. Create a sound set organized by timbres and notes (e.g., C3, D3, etc.).
  4. Upload the audio files into the appropriate folder structure.
  5. Ensure all paths are correct and validate functionality before saving changes.

Example JSON Structure:

{
  "nome": "My Environment",
  "sfondo": "path/to/background.jpg",
  "setSuoni": [
    {
      "nome": "Timbre 1",
      "suoni": {
        "C3": "path/to/C3.mp3",
        "D3": "path/to/D3.mp3"
      }
    }
  ]
}

Save the JSON file and refresh the application to load the new environment.
